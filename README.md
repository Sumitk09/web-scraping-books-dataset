# Books to Scrape Web Scraper

This project demonstrates web scraping using Python.

The scraper downloads HTML pages from the "Books to Scrape" website and extracts useful information such as:

- Book Title
- Price
- Star Rating
- Stock Availability

The extracted data is converted into a structured dataset and saved as a CSV file.

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas

## Project Workflow

1. Download HTML pages from the website
2. Parse the HTML using BeautifulSoup
3. Extract book details
4. Store the extracted data
5. Convert data into a CSV dataset
